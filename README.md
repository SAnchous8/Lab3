# Задание 1

### Текст задачи

Решить задачу из лабораторной работы №2 (Задание 1. List.map) для
последовательности

### Алгоритм решения

1. Попадаем в функцию main - точку входа программы

2. Запрашиваем у пользователя количество чисел для ввода:
   Выводим сообщение "Сколько чисел будете вводить?"
   Считываем введенное значение в переменную count

3. Создаем последовательность numbers, используя генератор последовательности:
   Запускаем цикл for от 1 до count
   Для каждого значения i вызываем функцию readNumber с аргументом i
   Результат каждого вызова добавляется в последовательность

4. В функции readNumber:
   Выводим приглашение "Число {index}:"
   Считываем введенное пользователем число в переменную input
   Проверяем корректность ввода:
   Если input меньше или равно нулю, выводим сообщение "Ошибка, число должно быть натуральным" и возвращаем -1
   Если input больше нуля, возвращаем введенное число

5. Применяем функцию minDigit к каждому числу из последовательности numbers:
   Для каждого числа вызывается функция minDigit

6. В функции minDigit:
   Проверяем, равно ли число нулю:
   Если n равно 0, сразу возвращаем 0
   Иначе вызываем вспомогательную рекурсивную функцию getDigits:
   Передаем в нее число n и начальное значение минимума 9
   В функции getDigits:
   Если num равно 0, возвращаем накопленное значение currentMin
   Иначе:
   Получаем последнюю цифру числа: digit равно num mod 10
   Находим минимум между digit и currentMin
   Рекурсивно вызываем getDigits с num деленным на 10 и новым минимумом

7. Выводим результаты:
   Выводим сообщение "Исходная последовательность:"
   Выводим результат применения minDigit ко всем числам

8. Завершаем программу с кодом возврата 0

# Задание 2

### Текст задачи

Решить задачу из лабораторной работы №2 (Задание 2. List.fold) для
последовательности

### Алгоритм решения

1. Попадаем в функцию main

2. Запрашиваем у пользователя количество чисел:

   Выводим сообщение: "Сколько чисел будете вводить?"

   Считываем введенное значение в значение count

3. Создаем список numbersList:

   Запускаем цикл for от 1 до count

   Для каждого значения i вызываем функцию readNumber i

   Добавляем полученное значение в список

4. В функции readNumber с параметром index:

      Выводим сообщение: "Число {index}: "

      Считываем введенное значение и возвращаем его

5. Возвращаемся в функцию main:

      Создаем последовательность numbersSeq из списка numbersList

      Запускаем цикл for по элементам numbersList

      Для каждого значения n добавляем его в последовательность

6. Выводим сообщение: "Исходный список:"

      Выводим содержимое списка numbersList

7. Запрашиваем у пользователя искомую цифру:

      Выводим сообщение: "Какую цифру ищем? "
   
      Считываем введенное значение в значение targetDigit

8. Проверяем корректность ввода:

      Если targetDigit меньше 0 или больше 9, выводим сообщение "Ошибка: цифра должна быть от 0 до 9"
   
      Если targetDigit от 0 до 9, используем Seq.fold для подсчета элементов
   
      Начальное значение аккумулятора равно 0
   
      Для каждого значения number в последовательности numbersSeq вызываем функцию containsDigit с аргументами number и targetDigit
   
      Добавляем результат вызова containsDigit к аккумулятору
   
      Сохраняем итоговое значение в result
   
      Выводим сообщение: "Количество элементов, содержащих цифру {targetDigit}: {result}"

9. В функции containsDigit с параметрами number и digit:

      Проверяем, равно ли number нулю:

      Если number = 0 и digit = 0, возвращаем 1
   
      Если number = 0 и digit != 0, возвращаем 0
   
      Если number > 0, вызываем функцию checkDigits с аргументом number

10. В функции checkDigits с параметром num:

      Проверяем базовый случай:

      Если num = 0, возвращаем 0

      Если num > 0, извлекаем последнюю цифру currentDigit из num

      Проверяем, равна ли currentDigit искомой цифре digit:

      Если равна, возвращаем 1

      Если не равна, вызываем checkDigits с num/10

11. Завершаем программу с кодом возврата 0
